Color Values

Color values in CSS can be specified using various formats, including hexadecimal (HEX), RGB, RGBA, HSL, HSLA, color names, or keyword values.

Syntax: color: value; color: #FF0000;

Examples:

  • HEX: #FF0000
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5)
  • HSL: hsl(0, 100%, 50%)
  • HSLA: hsla(0, 100%, 50%, 0.5)
  • Color Name: red
  • Keyword Value: inherit

Color Opacity

Description: Opacity property controls the transparency of an element, allowing you to see through it to varying degrees.

Syntax: opacity: value; (value ranges from 0 to 1)

Example: This pop-up has an opacity of 0.7.
