Text Properties

This page shows you the different ways you can style your texts!

×

Text Color

Text color property is used to define the color of the text content.

Syntax: color: value; color: red;

Example: This text is color red!

×

Text Decoration

Text decoration property is used to add decorations to text, such as underline, overline, line-through, or blink.

Syntax: text-decoration: value; text-decoration: underline;

Example: This is underlined text.

This is dotted underlined text.

This is red dotted underlined text.

This is overlined text.

×

Text Alignment

Text alignment property is used to specify the horizontal alignment of text content within its container.

Syntax: text-align: value; text-align: center;

Example:

This text is centered

This text is at the right

This text is at the left

×

Text Transformation

Text transformation property is used to control the capitalization of text, such as making it uppercase, lowercase, or capitalized.

Syntax: text-transform: value; text-transform: uppercase;

Example: i am an uppercase text
